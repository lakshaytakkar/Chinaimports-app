## Error and system messages

### Loading states
- Default: "Loading..."
- Specific: "Loading conversations..." / "Loading project details..." / "Submitting payment..."

### Errors
- Generic: "Something went wrong. Please try again."
- Network: "Can't connect. Check your internet and retry."
- Server: "We're having trouble on our end. Our team is fixing it."
- Auth expired: "Your session expired. Please sign in again."

### Confirmations
- Message sent: (silent — the send animation is enough feedback)
- Document uploaded: "Document shared" (toast, brief)
- Profile updated: "Profile saved" (toast, brief)
- Payment submitted: "Payment processing..." → "Payment successful" (modal, dismissible)

### Destructive confirmations
- Archive project: "Archive this project? You can unarchive from the filter menu."
- Delete message: "Delete this message? This can't be undone."
- Sign out: "Sign out of Suprans App? Your data stays safe."

## Microcopy for buttons and toasts

### Common buttons
- Continue → (for moving forward in multi-step flows)
- Next → (for form steps)
- Save (for form submissions with changes)
- Cancel (for abandoning an in-progress action)
- Close (for closing modals/screens)
- Done (for finishing a flow)
- Try again (for retry after error)
- Open (for opening an item)
- Start (for beginning an action — "Start conversation", "Start RFQ")

### Common toasts
- "Copied to clipboard"
- "Link opened"
- "Saved"
- "Sent"
- "Added"
- "Removed"
- "Document downloaded"

### Avoid these words on buttons
- "Submit" (except for formal RFQ submission — use "Send", "Continue", "Save" instead)
- "Click here" (this is a mobile app, don't say "click")
- "Please" (too passive)
- "OK" / "Cancel" on important confirmations (use specific verbs: "Delete" / "Keep")

## Voice: quick reference

Good: "Your project is in production. Next stage in 3 days."
Bad: "🎉 Awesome! Your amazing project is making great progress! 🚀"

Good: "This payment is overdue. Tap to settle."
Bad: "Oh no! Looks like you forgot to pay. No worries, we'll help you fix it!"

Good: "Sourcing three factory options now. Quote in 48 hours."
Bad: "Hang tight while we find the perfect partners for your dream project! ⏰"

Professional, calm, specific. Always.
